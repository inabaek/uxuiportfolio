UX/UI Projects

Project name: Get To Know You / Conceptual project

This is my first case study. Get To Know You is a question card game app that helps users have fun and quality time with their family or friends. It provides various topic options and helps users start a conversation with people. My role was to research, design, and test all UIs related to the project. I collaborated with another designer to launch this project.
